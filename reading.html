Recommended Reading

So who are you? This book really has two audiences. If you're a Java developer, then this book assumes that you're an intermediate Java developer with little or no experience with Bluetooth. You'll get the most out of this book if you've written a few Java classes on your own (especially J2ME applications).

If you’re a Bluetooth developer, then this book becomes useful to you after Chapter 2. If you've never used Java before, then we'd suggest that you read the first few chapters of a J2ME book before you read this book. If you fit in this category, then we recommend that you obtain Wireless Java: Developing with J2ME, Second Edition by Jonathan Knudsen.
